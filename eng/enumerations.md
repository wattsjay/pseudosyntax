## Declaration

> **NOTE**
> Type declarations use the `=` symbol, not `←` used by variable assignment.

```
  TYPE <identifier> = (<option_1>, <option_2>, <option_3>, ...)
```

## Using

1. Get all options as an array
```
  <enum_identifier>.options()
```
